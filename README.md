# pythaidate

(This is work-in-progress...)

Thailand has used several different calendar systems and date formats over the last millennium and
several are still in use today. This project is developing conversion software for the most important
calendars and eras including lunar calendar (ปฏิทินจันทรคติ), solar calendar (ปฏิทินสุริยคติ), Buddhist Era
(พุทธศักราช), Great Era (มหาศักราช), Minor Era (จุลศักราช), Rattanakosin Era (รัตนโกสินทร์ศก) as well as the
Western Gregorian and Julian calendars.

Limits
======

- ปฏิทินปักขคณนา
  Epoch: 1736-01-28 AD, 2279-01-28 BE, 2355148 Julian Date
  Pre-epoch: raises ValueError
